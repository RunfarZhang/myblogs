#### 相关命令

* ```dos
  net user 
  net user Adam
  net user Adam 1 # 给Adam修改账号密码
  net user abc 123 /add	# 修改密码或新建用户
  net user abc /del # 删除用户
  net user shimisi /active:no    # 禁用shimisi用户（封号）
  ```
